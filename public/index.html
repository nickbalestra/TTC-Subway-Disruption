TTC Subway Disruption

A simple tool to display, add and remove disruptions on the 4 metro lines in Toronto operated by the TCC. In the app state an array of disruptions is maintained. Each disruption has the following format 

disruption: {
  line: 1, 
  branch: 0, 
  notes: 'Closing all stretches of track',
  stops:[6, 7, 8, 9]
}
